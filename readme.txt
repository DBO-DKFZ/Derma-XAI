HOW TO SETUP

Run the following commands in the terminal:
	conda create -n python=3.9.9 env pip
	pip install -r requirements.txt


HOW TO RUN THE WORKING EXAMPLE 

Please start jupyter notebook and open train.ipynb to run the working example. Scroll down to reach the cell with the example.
A demo dataset of 50 images are available in the data/ directory.
It takes 2-3 minutes to train the model on the demo data and 1.1 seconds to get the prediction and explanations.


HOW TO REPRODUCE QUANTITATIVE RESULTS

The jupyter notebooks necessary to reproduce the quantiative results in the manuscript are available in the scripts/ directory,
and the data used for the quantitative results are available the data/ directory.
Some file paths might need to be changed to refer to files in the data/ directory.
